# MLH Fellowship Summer 2026 (Remote)

## Introduction
MLH Fellowship pairs fellows with maintainers and industry mentors to build production software remotely. This 2026 guide covers eligibility, timeline, and application flow. Dates are Expected/Indicative.

## Program Overview
- Remote pods with mentor guidance
- Ship OSS/features with code reviews
- Weekly demos and evaluations; stipends paid in tranches

## Eligibility
- Ability to work full-time remotely for 12 weeks
- Coding samples or OSS contributions
- English communication and async readiness

## How to Apply
1) Submit application with resume and code samples. 2) Complete interview/coding task. 3) Match to pod and confirm availability.

## FAQs
- Paid? Yes, region-based stipends (Expected/Indicative).
- Remote? Fully remote.
- Need prior OSS? Helpful but not required; projects demonstrating quality suffice.
